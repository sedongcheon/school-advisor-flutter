## 자세한 설명 (Play Console — 4000자 이내, App Store — 4000자 이내)

```
학교폭력, 어디서부터 시작해야 할지 막막했나요?
학폭 나침반은 절차·법령을 차분히 안내하고, 사안을 본인 기기에 노트로 정리하도록 도와드려요.

학폭위 관련 절차·법령을 익명으로 물어보고, 사안을 단계별로 정리해 둘 수 있어요. 회원가입이 필요 없고, 모든 대화·메모는 본인 기기에만 저장됩니다.

[ 누구를 위한 앱인가요 ]
- 직접 알아보고 정리하고 싶은 학생
- 자녀와 함께 차분히 살펴보고 싶은 보호자
- 학교에서 안내·교육 자료로 활용하려는 교사·전담기구

[ 핵심 기능 ]
■ 학폭위 자문 AI 챗봇
  - 학교폭력예방법을 근거로 답변
  - 답변마다 인용 조문 제공 — 탭하면 원문 확인
  - 모든 대화는 기기에만 저장 (익명)

■ 사안 노트 (개인용)
  - 3단계로 빠르게 작성 (역할 → 유형 → 시점·장소)
  - 메모 번호로 다시 열람
  - 실명·민감정보는 자동 마스킹
  - 단계 진행은 본인이 직접 옮기는 체크리스트

■ 법령·FAQ·절차 흐름도
  - 학폭예방법·시행령·시행규칙 검색
  - 학생·학부모가 가장 많이 묻는 질문
  - 절차 6단계 흐름도

■ 117 학교폭력 신고센터 바로 전화
  - 24시간 상담, 통화료 무료

[ 중요 — 신고 책임 분리 ]
■ 본 앱은 사안을 본인 기기에 정리하는 노트일 뿐, 신고를 학교·교육청·경찰에 자동으로 전달하지 않습니다.
■ 실제 신고·심의·조치는 학교 학교폭력 전담기구, 교육지원청, 또는 117 학교폭력 신고센터에 직접 연락해야 합니다.
■ 앱에 작성한 메모는 학교에 보낼 때 참고 자료로 사용할 수 있습니다.

[ 안전 약속 ]
- 회원가입·실명 입력 없음
- 익명 디바이스 식별자만 사용
- 모든 통신은 HTTPS 암호화
- 답변마다 면책 고지 자동 부착
- 입력한 실명·주민번호·휴대폰 번호는 자동 마스킹

[ 면책 고지 ]
본 앱이 제공하는 답변은 일반적인 정보 안내이며 법률 자문이 아닙니다.
실제 사건의 처리·구제는 학교, 교육청, 법률 전문가와 상담하세요.

긴급 상황 시 117(학교폭력 신고센터) 또는 112(경찰)로 즉시 연락하세요.
```

## 스크린샷 캡션 (각 5장 분량 한국어)

스크린샷은 **에뮬레이터(Pixel Fold API 35)** 또는 **iOS 시뮬레이터(iPhone 16)** 에서 캡처:

1. **홈 (학생 모드)**
   - 캡션: "차분히 시작해요. 무엇이 궁금한지 묻기만 하세요."

2. **챗봇 답변 + 인용 조문**
   - 캡션: "학교폭력예방법을 근거로 답변해 드려요. 인용 조문은 한 번 탭으로 확인."

3. **사안 노트 3-step**
   - 캡션: "역할·유형·시점만으로 사안 정리. 실명은 자동 마스킹. 본인 기기에 저장."

4. **사안 노트 단계 (4단계)**
   - 캡션: "정리→조사 메모→심의 메모→조치 메모. 단계는 본인이 직접 옮길 수 있어요."

5. **117 긴급 배너**
   - 캡션: "실제 신고는 117 또는 학교 전담기구. 24시간 상담, 통화료 무료."

6. (보너스) **알림 인박스**
   - 캡션: "사안 메모 단계 변경, 일정, 가이드 안내를 한 곳에서."

## "최근 업데이트" 출시 노트 (한국어, 500자 이내)

### v0.1.0 — 첫 공개
```
학폭 나침반의 첫 출시입니다.

- 학교폭력 절차·법령을 안내하는 AI 챗봇 (실시간 답변 + 인용 조문)
- 학생·보호자·교사 누구나 같은 도구로 활용
- 사안 노트 3단계 작성 + 메모 번호로 다시 열람 (본인 기기 저장)
- 학교폭력예방법·시행령 30+ 조문 검색
- 절차 6단계 흐름도
- 117 학교폭력 신고센터 바로 전화

※ 이 앱의 사안 노트는 본인 기기 메모이며, 실제 신고는 학교 전담기구 또는 117 로 연락해 주세요.

피드백은 설정 → 라이선스 → 문의로 보내주세요. 감사합니다.
```

### v0.x — 향후 업데이트 템플릿

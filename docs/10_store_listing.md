## 자세한 설명 (Play Console — 4000자 이내, App Store — 4000자 이내)

```
학교폭력, 어디서부터 시작해야 할지 막막했나요?
학폭 나침반은 신고부터 심의·조치까지 모든 단계를 차분히 안내합니다.

학폭위 관련 절차·법령에 대해 익명으로 물어보고, 신고서를 작성하고,
진행 상황을 추적할 수 있어요. 회원가입은 필요 없으며, 모든 대화는
기기에만 저장됩니다.

[ 누구를 위한 앱인가요 ]
- 직접 도움이 필요한 학생
- 자녀의 사안을 함께 살피는 보호자
- 사안을 처리하는 교사·전담기구

[ 핵심 기능 ]
■ 학폭위 자문 AI 챗봇
  - 학교폭력예방법을 근거로 답변
  - 답변마다 인용 조문 제공 — 탭하면 원문 확인
  - 모든 대화는 기기에만 저장 (익명)

■ 익명 신고서
  - 3단계로 빠르게 작성 (역할 → 유형 → 시점·장소)
  - R-번호로 진행 상황 조회
  - 실명·민감정보는 자동 마스킹

■ 진행 상황 추적
  - 접수 → 사안 조사 → 심의 → 조치 4단계
  - 단계 변경 시 알림 자동 적재
  - 보호자 모드는 자녀 진행 카드 한눈에

■ 법령·FAQ·절차 흐름도
  - 학폭예방법·시행령·시행규칙 검색
  - 학생·학부모가 가장 많이 묻는 질문
  - 신고부터 심의까지 6단계 절차도

■ 117 학교폭력 신고센터 바로 전화
  - 24시간 상담, 통화료 무료

[ 안전 약속 ]
- 회원가입·실명 입력 없음
- 익명 디바이스 식별자만 사용
- 모든 통신은 HTTPS 암호화
- 답변마다 면책 고지 자동 부착
- 입력한 실명·주민번호·휴대폰 번호는 자동 마스킹

[ 면책 고지 ]
본 앱이 제공하는 답변은 일반적인 정보 안내이며 법률 자문이 아닙니다.
실제 사건의 처리·구제는 학교, 교육청, 법률 전문가와 상담하세요.

긴급 상황 시 117(학교폭력 신고센터) 또는 112(경찰)로 즉시 연락하세요.
```

## 스크린샷 캡션 (각 5장 분량 한국어)

스크린샷은 **에뮬레이터(Pixel Fold API 35)** 또는 **iOS 시뮬레이터(iPhone 16)** 에서 캡처:

1. **홈 (학생 모드)**
   - 캡션: "차분히 시작해요. 무엇이 궁금한지 묻기만 하세요."

2. **챗봇 답변 + 인용 조문**
   - 캡션: "학교폭력예방법을 근거로 답변해 드려요. 인용 조문은 한 번 탭으로 확인."

3. **신고서 3-step**
   - 캡션: "역할·유형·시점만으로 익명 신고. 실명은 자동 마스킹."

4. **진행 상황 (4단계)**
   - 캡션: "접수→조사→심의→조치. 단계가 바뀌면 알림으로 알려드려요."

5. **117 긴급 배너**
   - 캡션: "긴급할 땐 117. 24시간 상담, 통화료 무료."

6. (보너스) **알림 인박스**
   - 캡션: "사안 진행, 일정, 가이드를 한 곳에서."

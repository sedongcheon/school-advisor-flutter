## 1. 본 서비스의 개인정보 수집 원칙

회사는 본 서비스를 **익명으로 이용 가능하도록** 설계했습니다. 회원가입·실명·이메일·전화번호 등 일반적인 개인 식별 정보를 **수집하지 않습니다**.

본 서비스가 자동으로 생성·수집하는 정보는 다음과 같습니다.

| 항목 | 내용 | 수집 시점 |
|---|---|---|
| 익명 디바이스 식별자 (UUID v4) | 앱 설치 시 클라이언트가 생성, 백엔드 호출 시 `X-Device-ID` 헤더로 전송 | 첫 실행 시 1회 |
| 질문 텍스트 | 사용자가 채팅창에 입력한 질문 본문 | 송신 시 |
| 답변·세션 기록 | 백엔드 RAG 시스템의 답변 + 인용 근거 | 답변 생성 시 |
| 사용량 카운트 | 일일 질문 수, 이용권 상태(plan / expires_at) | 채팅 송신·이용권 활성화 시 |
| 푸시 토큰 (FCM) | 알림 권한 동의 후 Firebase Cloud Messaging 토큰 | 사용자가 알림 토글을 켤 때 |

**다음은 수집하지 않습니다**: 이름, 주민등록번호, 휴대폰 번호, 이메일, 위치, 광고 식별자(ADID/IDFA), 연락처, 사진/카메라.

> 채팅창에서 사용자가 직접 입력한 실명·주민등록번호·학교명 등은 **클라이언트에서 자동 마스킹 다이얼로그가 노출**되며, 백엔드도 추가 마스킹 처리를 수행합니다.

## 2. 수집·이용 목적

수집된 정보는 다음 목적에만 사용됩니다.

1. **서비스 제공** — RAG 기반 답변 생성, 인용 조문 조회, 사용량 차감
2. **이용권 관리** — 무료/유료 플랜 한도, 만료 기한 관리
3. **품질 개선** — 답변 평가/신고 데이터를 통한 모델 튜닝
4. **알림 발송** — 법령 개정·중요 공지 (사용자가 명시적으로 토글 ON 한 경우만)
5. **장애 대응** — 서버 오류 진단, 보안 사고 대응
6. **법적 의무 이행** — 관련 법령에 따른 분쟁 해결

목적 외 이용은 없습니다. 광고/마케팅에 활용하지 않으며, 광고 식별자 추적도 하지 않습니다.

## 4. 제3자 제공 / 위탁

회사는 이용자의 정보를 **외부에 판매·임대·교환하지 않습니다**.

다만 서비스 운영을 위해 다음 처리 위탁을 진행합니다.

| 위탁받는 자 | 위탁 업무 | 처리 정보 |
|---|---|---|
| Google LLC (Firebase Cloud Messaging) | 푸시 알림 발송 | FCM 토큰 |
| Google LLC (Google Cloud Platform / Firebase) | 서버 인프라 | 익명 디바이스 식별자, 질문·답변 |

각 위탁사는 자체 개인정보 처리방침을 따릅니다. 위탁 범위 외 사용을 금지하는 계약을 체결합니다.

## 5. 이용자의 권리

이용자는 다음 권리를 행사할 수 있습니다.

- **열람·정정**: 본 앱은 익명이라 회원 정보 자체가 없습니다. 채팅 기록은 앱 내 "대화 이력 → 항목 삭제" 로 즉시 삭제 가능합니다.
- **삭제**: 앱을 제거하면 익명 디바이스 식별자가 무효화되며, 새 식별자가 발급됩니다.
- **처리 정지**: 알림 토글 OFF, 앱 제거 등으로 즉시 처리 정지 가능.
- **문의**: 위 연락처 이메일로 처리 내역 확인 요청.

## 6. 안전성 확보 조치

- 모든 통신은 **HTTPS / TLS** 로 암호화됩니다.
- 익명 디바이스 식별자는 클라이언트 기기의 **Secure Storage (Android Keystore / iOS Keychain)** 에 저장됩니다.
- 백엔드 데이터베이스는 인증된 직원만 접근 가능합니다.
- 사용자가 입력한 실명·주민등록번호·휴대폰 번호 등 민감 정보는 **자동 마스킹** 후 처리됩니다.

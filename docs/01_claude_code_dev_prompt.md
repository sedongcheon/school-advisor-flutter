# Claude Code 개발 지시 프롬프트 — Flutter 클라이언트

> **이 파일은 진실의 원천이 아닙니다.**
> 프로젝트 루트의 [`/CLAUDE.md`](../CLAUDE.md) 가 실제로 Claude Code 가 읽는 파일이며, 본 docs/01 은 동일 내용의 사본을 두 곳에 두며 표류시키지 않기 위해 **가벼운 포인터**로만 유지합니다.

## 사용법

새 작업/세션 시작 시:

```bash
claude
```
첫 메시지:
```
CLAUDE.md 읽고 진행해줘.
```

원본 프롬프트의 모든 내용 — 프로젝트 개요, 기술 스택, 디렉토리 구조, 안전장치, 개발 순서, 금기 사항 — 은 [`/CLAUDE.md`](../CLAUDE.md) 에서 확인하세요.
